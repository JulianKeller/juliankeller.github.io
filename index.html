Make Repo

Make Repo is a simple Shell script which creates a new local git repository and pushes it to GitHub. It will create a blank README.md and .gitignore automatically. It will also prompt the user and ask if they would like to create any other blank files. While creating a new repo is easy, there are several steps to take. This script makes it even easier by reducing it to a single step of running the script.
